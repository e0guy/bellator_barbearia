# 💈 Sistema de Barbearia

Sistema completo de agendamento para barbearia.

## 🚀 Funcionalidades
- Login e cadastro
- Agendamento de serviços
- Escolha de barbeiro
- Painel admin

## 🛠 Tecnologias
- Java + Spring Boot
- HTML, CSS, JavaScript
